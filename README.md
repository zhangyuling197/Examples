# Examples_Stealthy_Sensor_Attacks_for_Violating_Detectability_of_DES
Supplementary Materials.zip contains the MATLAB codes, in which the function NFA2DFA.m is taken from the folder “PetriBaR” by Y. Tong et al.
Before running the codes in Supplementary Materials.zip, please add the “PetriBaR” folder to the MATLAB path. 
The PetriBaR MATLAB toolbox is available at the following link: https://www.researchgate.net/publication/321431345_MATLAB_Toolbox_for_Petri_Nets
